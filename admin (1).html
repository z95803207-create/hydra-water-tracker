🔥 Firebase Setup Guide
Connect real-time analytics in 5 minutes
⚡ Why Firebase?
Firebase is Google's free backend service. With the Spark (free) plan you get 50,000 reads/day, 20,000 writes/day — more than enough for Hydra tracker.
1
Create Firebase Project
Go to console.firebase.google.com → Click "Add Project" → Name it hydra-tracker → Disable Google Analytics (not needed) → Create
2
Add Web App & Get Config
Project Settings → "Add App" → Web (</>) → Register → Copy the firebaseConfig object
3
Enable Firestore Database
Firestore Database → Create Database → Start in Test Mode → Choose region → Done
4
Paste Your Firebase Config Below
Fill in from your Firebase Console → Project Settings → Your Apps
API Key
Auth Domain
Project ID
Storage Bucket
Messaging Sender ID
